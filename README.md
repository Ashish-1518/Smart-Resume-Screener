# 📄 Smart Resume Screener

An AI-powered resume screening tool that ranks candidates based on job description relevance using **TF-IDF Vectorization** and **Cosine Similarity**.

## 🎯 Features

- 📝 Upload job description (paste text or upload .txt file)
- 📄 Upload multiple resumes (PDF, DOCX, TXT formats)
- 🏆 Get ranked results with match percentages
- 🔑 Extract key skills from job descriptions
- 📊 Visual candidate ranking
- ✅ Shortlist candidates above 50% match

## 🛠️ Tech Stack

- **Python** - Core programming language
- **Streamlit** - Interactive web interface
- **scikit-learn** - TF-IDF vectorization and cosine similarity
- **PyPDF2** - PDF text extraction
- **python-docx** - DOCX text extraction
- **pandas** - Data manipulation
